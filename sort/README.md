## Утилита sort

```sh
$ echo -ne "qwe\nasd\nzxc\n" | perl sort.pl -r -u
zxc
qwe
asd
```

## Основное

Поддержать ключи
* -n — сортировать по числовому значению
* -r — сортировать в обратном порядке
* -u — не выводить повторяющиеся строки
